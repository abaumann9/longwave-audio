# Contributing to LongWave Audio

Thanks for your interest in contributing! This project is in early development and I welcome help in all areas.

## Ways to Contribute

- **Hardware** - PCB design improvements, enclosure designs, assembly documentation
- **Firmware** - snapclient modifications, HaLow driver integration, audio quality improvements
- **Server** - Snapserver configuration, additional audio source integrations
- **Documentation** - Build guides, setup instructions, troubleshooting tips
- **Testing** - Range testing, latency measurements, compatibility reports

## Getting Started

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-improvement`)
3. Make your changes
4. Test your changes thoroughly
5. Commit with a clear message describing what and why
6. Open a pull request
